# seam-core
SeamCore is a Godot 4 addon that brings seamless, persistent worlds to your 3D games. Inspired by classic level streaming systems like Half-Life 2, it allows scenes to remember everything the player has changed—from physics objects and NPCs to triggers and world states.

With SeamCore, levels behave like connected parts of a single continuous world. Players can move forward, backtrack, and even carry objects across scene boundaries, all while the game accurately restores every interaction and change. It’s designed for developers who want immersive, stateful environments without building a custom save system from scratch.
